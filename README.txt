BTX MOSTRUÁRIO — COMO USAR (PC + GITHUB PAGES)

1) Estrutura
btx-mostrario/
  index.html
  estilos.css
  app.js
  ativos/
    orlando.jpg
    btx-logo.png

2) Sua foto
Coloque SUA FOTO dentro de: ativos/orlando.jpg

3) Logo BTX
Coloque o logo dentro de: ativos/btx-logo.png

4) Teste no PC
Clique duas vezes no index.html e abra no Chrome.
Se não aparecer a foto/Logo: confira nomes e extensões.

5) GitHub Pages
- Suba tudo pro repositório (inclusive a pasta ativos)
- Settings > Pages > Branch: main / root
- Abra o link do Pages e faça Ctrl+F5

Pronto.
